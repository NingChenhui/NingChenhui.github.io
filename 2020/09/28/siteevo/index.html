开一篇记录一下网站的进化史~
我会把一些部件的实现方式记录在这哦。坑都帮你踩好了
有更好的实现方式也可以跟我交流~

2020年

11月1日

  • 恢复Marked渲染，折腾好久：解决Mathjax与代码高亮的冲突。

10月23日

  • 支持$LaTeX$:使用 Mathjax。 好像只能强制全局渲染，关不掉了……

10月5日

  • 图片加载问题：解决Github网页上图片显示失败的问题。需要各位手动设置……（反正我现在访问没啥问题了

10月4日

  • 网站主页：目前是仿着11岁时的网站搭的页面~ 使用了主题里的py模板。
  • 置顶文章：参考hexo博客优化之文章置顶+置顶标签。不过这篇是next主题的，Nexmoe需要在\\layout\\_partial\\_post\\meta.ejs里的<div class="nexmoe-post-meta nexmoe-rainbow"> 和 \\layout\\index.ejs里的<div class="nexmoe-post-meta nexmoe-rainbow">中插入
      <% if (page.top){ %>
     <a> <i class="nexmoefont icon-right"></i>
      置顶</a>
  <% } %>

9月28日

  • 网站访问统计：不蒜子计数 这个主题的话可以把代码分别插在\\layout\\_partial\\sidebar.ejs，\\layout\\_partial\\copyright.ejs和\\layout\\layout.ejs里。
  • 文章加密：hexo-blog-encrypt 加密后有些部件失效了就，包括音乐播放、导航。脑壳痛
  • 看板娘：教程 两行代码同样插在\\layout\\layout.ejs里。 位置在waifu.css里调，left改成right

9月27日

  • 个人信息界面
    图片并排实现：查了不少方法都不行，最后发现直接用表格再插图就好（有够偷懒
    隐藏标签：使用 <details> 元素 貌似IE不支持
    封面图片：这东西用相对路径的话主页就不显示图片，于是直接用了在上传网站后Github里存放图片的网络位置。

9月26日

  • 开始搭建博客

    1. 建站教程：用 Hexo 和 GitHub Pages 搭建博客 另外还可参考如何用github page搭建博客？
    2. Hexo文档：文档 | Hexo
    3. 网站模板：Nexmoe’s Theme 唔分类栏好像有点问题……
    4. 主题开发者的blog：折影轻梦 (以后有钱了我也要买个域名
    5. 博客编写：
      • Markdown教程 (太菜了啥都不会
      • Sublime Markdown插件安装
    6. 一些命令：
       $ hexo new page "新文章"
 $ hexo clean
 $ hexo g
 $ hexo g -d
 $ hexo s
      这里 $ hexo d 有可能会报错Please tell me who you are，解决办法

  • 评论系统
    这里我用的是Disqus（需科学），参考Nexmoe 评论系统的Disqusjs，apikey是配置 Disqus Application 时获取的公钥。

  • 赞助页面 觉得有用的话就赏点吧 孩子可穷了
    本来没想搞的，炜神建议搞一个
    插入音频：hexo-tag-aplayer
    网站的图标库：IconFont Demo
